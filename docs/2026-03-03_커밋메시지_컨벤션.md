# Commit Message Convention

## 커밋 메시지 컨벤션

### 형식

```text
TYPE[S14P21E106-번호] 간결한 설명

(선택) 본문 - 상세 설명
```

**⚠️ Jira Task 키를 사용합니다. GitLab 이슈 번호(#)가 아닙니다.**

### 타입

| **타입** | **설명** | **예시** |
| --- | --- | --- |
| `FEAT` | 새로운 기능 | `FEAT[S14P21E106-10] 사용자 로그인 기능 추가` |
| `FIX` | 버그 수정 | `FIX[S14P21E106-15] 로그인 시 NPE 수정` |
| `DOCS` | 문서 변경 | `DOCS[S14P21E106-3] API 문서 업데이트` |
| `STYLE` | 코드 포매팅 (동작 변경 없음) | `STYLE[S14P21E106-20] 코드 정렬` |
| `REFACTOR` | 리팩토링 | `REFACTOR[S14P21E106-25] UserService 메서드 분리` |
| `TEST` | 테스트 추가/수정 | `TEST[S14P21E106-30] UserService 단위 테스트 추가` |
| `CHORE` | 빌드, 설정 변경 | `CHORE[S14P21E106-5] Gradle 의존성 업데이트` |
| `HOTFIX` | 긴급 수정 | `HOTFIX[S14P21E106-99] 인증 토큰 만료 이슈 수정` |

### 규칙

- 타입은 **대문자**
- 제목은 50자 이내
- 제목은 명령형 ("추가한다" 아닌 "추가")
- 본문은 72자에서 줄바꿈
