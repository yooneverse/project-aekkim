# 구독 API

## 공통 규칙

- Base URL: `/api/v1`
- 공통 응답 형식: `ApiResponse<T>`
- 이 문서는 가맹점 매핑, 서비스/번들 조회, 구독 관리, 체크인, 사용량 API를 함께 다룬다.

---

## 1. 가맹점 매핑 조회 (배치)

- Method: `POST`
- URL: `/api/v1/merchant-mappings/batch-lookup`
- 인증: 필요
- 설명: 가맹점 원문과 AI 예측 결과를 기준으로 서비스 자동 매칭 결과를 일괄 조회한다.

### Request Body

| 필드명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `items` | array(object) | Y | 매핑 조회 요청 목록 |
| `items[].merchantRaw` | string | Y | 가맹점 원본 문자열 |
| `items[].predictedServiceId` | long | Y | AI 예측 서비스 ID |
| `items[].predictedServicePlanId` | long | Y | AI 예측 요금제 ID |

### Response Data

| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| `results` | array(object) | 매칭 결과 목록 |
| `results[].merchantRaw` | string | 가맹점 원문 |
| `results[].matched` | boolean | 자동 매칭 여부 |
| `results[].serviceId` | long | 매핑된 서비스 ID |
| `results[].servicePlanId` | long | 매핑된 요금제 ID |
| `results[].hitCount` | int | 누적 적중 횟수 |

---

## 2. 가맹점 매핑 확정

- Method: `POST`
- URL: `/api/v1/merchant-mappings/batch-confirm`
- 인증: 필요
- 설명: 사용자가 선택한 서비스/요금제로 구독을 생성하고 가맹점 매핑을 누적한다.

### Request Body

| 필드명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `items` | array(object) | Y | 가맹점 매핑 확정 요청 목록 |
| `items[].merchantRaw` | string | Y | 가맹점 원문 |
| `items[].serviceId` | long | Y | 확정 서비스 ID |
| `items[].servicePlanId` | long | Y | 확정 요금제 ID |
| `items[].nextBillingDate` | string(date) | N | 다음 결제일 |

### Response Data

| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| `results` | array(object) | 생성 결과 목록 |
| `results[].subscriptionId` | long | 생성된 구독 ID |
| `results[].serviceId` | long | 서비스 ID |
| `results[].serviceName` | string | 서비스명 |
| `results[].servicePlanId` | long | 요금제 ID |
| `results[].monthlyPrice` | int | 월 구독 금액 |
| `results[].nextBillingDate` | string(date) | 다음 결제일 |
