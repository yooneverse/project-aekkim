# 프로모션 API

## 공통 규칙

- Base URL: `/api/v1/promotions`
- 공통 응답 형식: `ApiResponse<T>`
- 모든 API는 `Authorization: Bearer {accessToken}` 헤더가 필요하다.

---

## 1. 추천 프로모션 조회

- Method: `GET`
- URL: `/api/v1/promotions/recommendations`
- 설명: 사용자의 구독 상태와 최근 사용 이력을 기준으로 카테고리별 추천 프로모션을 조회한다.

### Query Parameter

| 변수명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `promotionType` | string | N | `ALL`, `BUNDLE`, `CARD_BENEFIT`, `PROMO`. 기본값 `ALL` |
| `page` | int | N | 현재 구현에서는 유효성 검증만 수행한다. 기본값 `0` |
| `size` | int | N | 카테고리별 최대 노출 개수. 기본값 `3`, 최대 `20` |

### Response Data

| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| `categories` | array(object) | 카테고리별 추천 목록 |
| `categories[].category` | string | 서비스 카테고리 |
| `categories[].bundles` | array(object) | 번들 추천 목록 |
| `categories[].promotions` | array(object) | 일반 프로모션 추천 목록 |
| `categories[].cardBenefits` | array(object) | 카드 혜택 추천 목록 |
| `categories[].*.promotionId` | long | 프로모션 ID. 번들은 음수 ID로 표현 |
| `categories[].*.promotionType` | string | `BUNDLE`, `PROMO`, `CARD_BENEFIT` |
| `categories[].*.title` | string | 제목 |
| `categories[].*.summary` | string | 요약 문구 |
| `categories[].*.originalPrice` | int | 할인 전 가격 |
| `categories[].*.discountPrice` | int | 할인 후 가격 |
| `categories[].*.billingCycle` | string | 결제 주기 |
| `categories[].*.sourceUrl` | string | 외부 링크 |
| `categories[].*.imageUrl` | string | 이미지 URL |
| `categories[].*.recommendationScore` | int | 추천 점수 |
| `categories[].*.recommendationReasons` | array(string) | 추천 사유 |
| `categories[].*.startsAt` | string(datetime) | 시작 시각 |
| `categories[].*.endsAt` | string(datetime) | 종료 시각 |
| `categories[].*.services` | array(object) | 연관 서비스 목록 |
| `size` | int | 응답에 사용한 카테고리별 최대 노출 개수 |

## 2. 프로모션 상세 조회

- Method: `GET`
- URL: `/api/v1/promotions/{promotionId}`
- 설명: 특정 프로모션 또는 번들 추천의 상세 정보를 조회한다.

### Path Variable

| 변수명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `promotionId` | long | Y | 프로모션 ID. 번들 추천은 음수 ID 사용 |

### Response Data

| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| `promotionId` | long | 프로모션 ID |
| `promotionType` | string | `BUNDLE`, `PROMO`, `CARD_BENEFIT` |
| `title` | string | 제목 |
| `summary` | string | 요약 문구 |
| `originalPrice` | int | 할인 전 가격 |
| `discountPrice` | int | 할인 후 가격 |
| `billingCycle` | string | 결제 주기 |
| `sourceUrl` | string | 외부 링크 |
| `imageUrl` | string | 이미지 URL |
| `startsAt` | string(datetime) | 시작 시각 |
| `endsAt` | string(datetime) | 종료 시각 |
| `services` | array(object) | 연관 서비스 목록 |

### 비고

- 번들 상세는 내부적으로 `Bundle` 정보를 기반으로 계산되며, 응답에서는 `promotionType = BUNDLE`로 표현한다.
