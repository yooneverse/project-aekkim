# Issue/MR Title Convention

## 이슈 제목 컨벤션

### 형식

```
{파트}[{타입}] 간결한 설명
```

### 예시

| 파트 | 타입 | 예시 |
| --- | --- | --- |
| `BE` | `FEAT` | `BE[FEAT] 사용자 로그인 기능 추가` |
| `BE` | `FIX` | `BE[FIX] 로그인 시 NPE 수정` |
| `FE` | `FEAT` | `FE[FEAT] 로그인 페이지 구현` |
| `FE` | `FIX` | `FE[FIX] 버튼 클릭 이벤트 미동작 수정` |
| `BE` | `REFACTOR` | `BE[REFACTOR] UserService 메서드 분리` |
| `FE` | `CHORE` | `FE[CHORE] ESLint 설정 추가` |

---

## MR 제목 컨벤션

### 형식

```
{타입}[{JIRA-KEY}] 간결한 설명
```

### 예시

```
FEAT[S14P21E106-42] 사용자 로그인 기능 추가
FIX[S14P21E106-57] 로그인 시 NPE 수정
REFACTOR[S14P21E106-63] UserService 메서드 분리
CHORE[S14P21E106-10] Docker 환경 구성
```

**⚠️ GitLab 이슈는 더 이상 생성하지 않습니다. 이슈는 Jira에서 관리합니다.**
