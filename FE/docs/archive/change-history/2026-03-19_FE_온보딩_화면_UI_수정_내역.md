# 2026-03-19 FE 온보딩 화면 UI 수정 내역

## 목적

- 제공 목업 기준으로 온보딩 화면의 시각적 밀도, 정렬, 타이포그래피, 소셜 로그인 버튼 아이콘을 재정비한다.

## 최종 결과 요약

- 온보딩 텍스트 + 버튼 블록을 화면 중심 기준으로 배치했다.
- 제목/본문의 자간, 줄간격, 굵기를 목업 톤에 맞게 조정했다.
- 모든 텍스트를 중앙 기준으로 정렬되도록 레이아웃을 보정했다.
- 소셜 로그인 버튼의 텍스트 아이콘(G/K/N)을 실제 에셋 로고로 교체했다.
- 버튼 내 텍스트가 버튼 중심선 기준으로 정확히 중앙에 오도록 패딩/정렬 구조를 수정했다.

## 수정 파일

### 1) 온보딩 화면 레이아웃

- `FE/app/src/main/java/com/ssafy/e106/feature/auth/ui/OnboardingScreen.kt`
  - `AuthFlowScaffold`를 `centerBody = true`로 구성.
  - 본문(`body`)에 히어로 텍스트와 소셜 버튼을 함께 배치해 하나의 블록으로 중앙 정렬.
  - 텍스트와 버튼 그룹 사이 간격을 `20.dp`로 유지.
  - 버튼 그룹 내부 간격은 `Column(verticalArrangement = Arrangement.spacedBy(12.dp))`로 고정.
  - 기존 로고 이미지 섹션 제거 상태 유지.

### 2) 히어로 텍스트 정렬/간격

- `FE/app/src/main/java/com/ssafy/e106/feature/auth/ui/component/AuthHeroSection.kt`
  - 제목/부제 `Text`에 `Modifier.fillMaxWidth()`를 적용해 중앙 정렬 기준을 명확화.
  - 제목과 부제 간 간격을 `spacing.space4`로 유지.

### 3) 타이포그래피 스케일 조정

- `FE/app/src/main/java/com/ssafy/e106/core/ui/theme/Type.kt`
  - `displayLarge`
    - `fontWeight = FontWeight.Black`
    - `fontSize = 36.sp`
    - `lineHeight = 50.sp`
    - `letterSpacing = (-0.5).sp`
  - `bodyLarge`
    - `letterSpacing = 0.sp`

### 4) 소셜 버튼 아이콘/중앙 정렬 구조

- `FE/app/src/main/java/com/ssafy/e106/feature/auth/ui/component/SocialLoginButton.kt`
  - 텍스트 아이콘(`iconText`) 방식 제거, `@DrawableRes iconRes`로 변경.
  - `Button`에 `contentPadding = PaddingValues(0.dp)` 적용.
  - 내부 컨테이너를 `Box(weight(1f).fillMaxHeight(), contentAlignment = Alignment.Center)`로 변경.
  - 텍스트는 버튼 중앙, 아이콘은 좌측 고정(`align(CenterStart) + padding(start = 20.dp)`)으로 배치.
  - provider 매핑:
    - Google -> `R.drawable.ic_logo_google`
    - Kakao -> `R.drawable.ic_logo_kakao`
    - Naver -> `R.drawable.ic_logo_naver`

## 신규 리소스 파일

- `FE/app/src/main/res/drawable/ic_logo_naver.xml`

## 에셋 출처 및 매핑

- 원본 에셋 확인 경로
  - `FE/app/src/main/assets/google.svg`
  - `FE/app/src/main/assets/KakaoTalk_logo.png`
  - `FE/app/src/main/assets/naver.svg`
- Android Compose 사용을 위해 `res/drawable` Vector Drawable(XML) 리소스로 반영.

## 정렬 기준(최종)

- 화면 중앙(가로/세로) 기준: 온보딩 텍스트 + 버튼 묶음 블록 중심 정렬.
- 텍스트 중앙 기준: 제목/부제 모두 전체 폭 기준 중앙 정렬.
- 버튼 중앙 기준: 버튼 내부 텍스트가 버튼 중심선에 오도록 정렬.
