# 대시보드 로고 이미지 적용

> **작성일**: 2026-03-17  
> **범위**: `feature/dashboard/ui/DashboardScreen.kt` + `res/drawable-*/`

---

## 1. 개요

대시보드 화면 좌상단의 플레이스홀더 텍스트("AE")를 **애낌(AEKKIM) 브랜드 로고 이미지**로 교체했다.  
원본 이미지(1920x1080 PNG)를 Android 밀도 버킷별로 리사이징하고 WebP 포맷으로 변환하여 최적화했다.

### 핵심 변경 요약

- "AE" 텍스트 → 로고 이미지(`Image` + `painterResource`) 교체
- 밀도별 5단계 리소스 생성 (mdpi ~ xxxhdpi)
- WebP 변환 + 트리밍으로 총 용량 ~24KB (밀도 5벌 합계)

---

## 2. 이미지 최적화 상세

### 원본

| 항목 | 값 |
|---|---|
| 소스 | `애낌애낌.png` (금색 O + >> 화살표 로고) |
| 해상도 | 1920 x 1080 px |
| 포맷 | PNG (RGB) |
| 비고 | 흰색 배경. 앱의 밝은 배경과 자연 블렌딩 |

### 최적화 과정

1. **여백 트리밍**: 1920x1080 → 305x157 (sharp `trim`, threshold=20)
2. **밀도별 리사이징**: 40dp 기준 높이 맞춤, Lanczos 리샘플링
3. **WebP 변환**: quality=90, lossy 압축

### 결과

| 밀도 버킷 | DPI 배율 | 해상도 | 용량 |
|---|---|---|---|
| `drawable-mdpi` | 1x | 78 x 40 px | 1.8 KB |
| `drawable-hdpi` | 1.5x | 117 x 60 px | 2.8 KB |
| `drawable-xhdpi` | 2x | 155 x 80 px | 3.8 KB |
| `drawable-xxhdpi` | 3x | 233 x 120 px | 6.1 KB |
| `drawable-xxxhdpi` | 4x | 311 x 160 px | 8.6 KB |
| **합계** | | | **~23.1 KB** |

## 4. 코드 변경 diff

```kotlin
// Before
Surface(
    modifier = Modifier.size(40.dp),
    shape = MaterialTheme.shapes.large,
    color = MaterialTheme.colorScheme.surfaceVariant,
    border = BorderStroke(1.dp, MaterialTheme.colorScheme.outline),
) {
    Box(contentAlignment = Alignment.Center) {
        Text(
            text = "AE",
            style = MaterialTheme.typography.labelLarge,
            color = MaterialTheme.colorScheme.primary,
        )
    }
}

// After
Image(
    painter = painterResource(id = R.drawable.logo_aekkim),
    contentDescription = "애낌 로고",
    modifier = Modifier.height(40.dp),
    contentScale = ContentScale.Fit,
)
```

### 추가된 import

```kotlin
import androidx.compose.foundation.Image
import androidx.compose.ui.layout.ContentScale
import androidx.compose.ui.res.painterResource

```

## 5. 로고 교체 시 참고사항

- 리소스 이름: `logo_aekkim` (모든 밀도 폴더에 동일 이름)
- 로고 교체 필요 시 원본 이미지를 동일한 최적화 과정으로 처리하여 5개 밀도 폴더에 배치
- 로고 비율이 약 2:1 (가로:세로)이므로 `ContentScale.Fit` + `height` 제약으로 비율 유지
- 원본 RGB 이미지의 흰색 배경은 앱의 밝은 테마 배경과 자연스럽게 블렌딩됨
