E106 Demo Admin Console

서비스, 사용자, 구독, 번들, 프로모션 데이터를 조회하고 seed import를 확인할 수 있는 개발용 관리자 페이지입니다.

서비스 / 요금제 JSON Import

서비스 카탈로그 JSON 1개를 업로드해 서비스와 요금제를 함께 upsert합니다.

POST /api/v1/dev/admin/imports/service-catalog

서비스 / 요금제 조회

현재 저장된 서비스와 요금제 목록을 조회합니다.

GET /api/v1/dev/admin/services
ID code name category plans

사용자 조회

현재 저장된 사용자 목록을 조회합니다.

GET /api/v1/dev/admin/users
ID provider providerUserId email displayName alerts optionalConsent connectedAt lastLoginAt

구독 조회

현재 저장된 구독을 userId 기준으로 조회합니다.

GET /api/v1/dev/admin/subscriptions?userId=1
ID userId type service plan bundle nextBillingDate

번들 JSON Import

번들 카탈로그 JSON 1개를 업로드해 번들과 포함 서비스를 upsert합니다.

POST /api/v1/dev/admin/imports/bundles

번들 조회

현재 저장된 번들 마스터와 포함 서비스 목록을 조회합니다.

GET /api/v1/dev/admin/bundles
ID code name planName price services

프로모션 JSON Import

프로모션 JSON 파일을 여러 개 업로드해 일반 프로모션과 카드 혜택을 함께 upsert합니다.

POST /api/v1/dev/admin/imports/promotions

프로모션 조회

현재 저장된 프로모션과 연결 서비스 목록을 조회합니다.

GET /api/v1/dev/admin/promotions
ID type title billingCycle period services

구독 사용량 JSON Import

서비스 코드, 사용일, 사용분이 담긴 JSON 1개를 업로드해 일별 사용량 데이터를 upsert합니다.

POST /api/v1/dev/admin/imports/subscription-usages

구독 사용량 조회

현재 저장된 일별 사용량 원자료를 조회합니다.

GET /api/v1/dev/admin/subscription-usages
ID userId service usageDate usedMinutes

결제내역 JSON Import

사용자 결제내역 JSON 1개를 업로드해 결제 데이터를 저장합니다.

POST /api/v1/dev/admin/payment-histories

결제내역 조회

현재 저장된 결제내역 목록을 조회합니다.

GET /api/v1/dev/admin/payment-histories
ID paymentId merchantRaw amount paymentDate category

알림 전송 테스트

FCM 알림 전송 API를 직접 호출해 CHECKIN, PROMO, CHURN_REVIEW 알림을 테스트합니다.

POST /api/v1/dev/admin/notifications

배치 실행

타겟 선정과 알림 전송 배치를 직접 실행합니다.
